Базовые консольные команды git

Лучше всего изучать базовые консольные команды по главам 2 и 3 официальной книги по git [Rus]

Консольные команды позволяют сделать куда больше, чем приложения с графическим интерфейсом и иногда могут спасти жизнь работу последних нескольких часов.

Кроме этого при использовании удаленного доступа, например SSH, нет возможности использования графического интерфейса, а значит консольные команды - единственное, что остается.
