Подготовка

Перед изучением материалов стоит установить на компьютер: - GitHub desktop; - VS Code;

А также создать аккаунт на: - GitHub.

Git book

Начать изчение Git лучше всего с прочтения введения официальной книги по git [Rus]. Тут объясняется, что вообще такое git и зачем он нужен. Устанавливать git на компьютер как отдельную программу не надо.

GitHub

Для изучения GitHub лучше всего воспользоваться их собственными курсами на сайте - GitHub Skills [Eng]. Для изучения самых основ нужно пройти 2 курса: 1. Introduction to GitHub; 2. Communicate using Markdown;

Курсы представляют из себя шаблоны репозиториев. Инструкция по выполнению курса находится в описании к репозиторям.

GitHub desktop

  • видео [15:29] [Eng] о том, как пользоваться большинством базовых функций приложения GitHub desktop;

Решение Merge конфликтов в VS Code

  • видео [3:02] [Eng] объясняющее как решать merge конфликты с использованием VS Code. В видео показываются и другие моменты, но они не важны для самых новичков. В момент, когда у вас появятся merge конфликты GitHub desktop предложит выбрать приложение для их решения, нужно выбираться VS Code.

Результат

На данном этапе у тебя должно появиться понимание о следующих вопросах: - что такое система контроля версий; - зачем она нужна; - что такое git; - что такое commit; - что такое branch; - что такое pull-request; - как создавать проект на GitHub; - как создавать проект с использованием GitHub desktop; - как клонировать проект из интеренета с помощью GitHub desktop; - как делать комиты в GitHub desktop; - как создавать ветки и переключаться между ними в GitHub desktop; - как совмещать ветки и решать конфликты при совмещении с исользованием GitHub desktop; - как создавать и закрывать pull-request-ы на GitHub.

Этих знаний будет вполне достаточно, чтобы делать базовые вещи в git.

Если ты не пишешь код или документацию к нему, то изучать git на более глубоком уровне не обязательно. Но если тебе хочется, или нужно, то продолжить изучение можно, читая наши [[Git intermediate materials| базовые материалы]].
